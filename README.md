# Gleam x Raylib

Let's try to use Raylib inside Gleam.

Experimentation with Erlang and Gleam NIF (Native Implemented Functions), using [Raylib](https://github.com/raysan5/raylib).

![Demo](https://github.com/Zenor27/gleam_raylib/blob/main/gleam_raylib_demo.png?raw=true)

## Quick start

```console
$ make
$ gleam run
```
